Quick Start Tutorial

Map - Basemaps

Click the Map Tools Button and select Basemaps option.

Map - View Layers

Click the Map Tools Button and select View Layers option. You can control the visibility of layers on the map as well as change layer opacity by adjusting the opacity slider.

Map – Parking theme

Search for a Study Area by Name

Click on a Parking Feature to view detailed data in the Parking Inspector.

There are several Parking themes to choose from. You can inspect features in each parking theme. If you click on Weekday and Weekend Occupancy themes you will notice a Time Period control appears. You could choose Time of Day to view occupancy data by time of day data was collected.

For Peak Occupancy Parking you will notice an On/Off Street Parking control appears. You could choose On Street, Off Street or Both On/Off Street parking.

Map – Navigation

Choose Navigation Tips under Map Tools menu to see how you can navigate the map, zoom in a level and out a level, pan and recenter.

Map – Printing

Choose Navigation Tips under Map Tools menu to see how you can navigate the map, zoom in a level and out a level, pan and recenter.

Pan the map area that you want to print into the window.

Click Print to see Print Preview of your map.

Map – Parking Data

Click the Parking data option in the top navigation bar and select Summary.

Choose a Study Area and a Collection Year from the drop-down menu, use on and off switches to select what data you would like to view and click Show Data.

You can Inspect Parking data summary. Data is summarized by study area. Click Print at the top right corner of the data summary to print your data view.

Print Preview window comes up.

Click Show Map to view the map of the selected study area. Click Download Data to download Excel spreadsheet of the Data Summary by study area.

Select Inventory option under Parking Data in the top navigation bar. Select Study Area, Parking Type and a Collection Year. Click download data to download data summarized at blockface and facility level.

Select Occupancy option under Parking Data in the top navigation bar. Select Study Area, Parking Type and a Collection Year. Click download data to download data summarized at blockface and facility level.
